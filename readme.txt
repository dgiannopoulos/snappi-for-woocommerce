== Description ==

Το πρόσθετο Snappi Pay Later προσθέτει στην ιστοσελίδα σας την πύλη πληρωμών της Snappi για το WooCommerce.

Με τη μεταφόρτωση του πρόσθετου θα λάβετε ένα συμπιεσμένο αρχείο το οποίο θα εγκαταστήσετε στην ιστοσελίδα σας. Όταν εγκατασταθεί και ενεργοποιηθεί με επιτυχία το πρόσθετο, θα δημιουργηθεί μία νέα σελίδα στις ρυθμίσεις πληρωμών του WooCommerce. Η σελίδα ονομάζεται Snappi Pay Later και θα τη βρείτε μέσα στην καρτέλα Ταμείο.

Αρχικά, πρέπει να ενεργοποιήσετε την πύλη των πληρωμών, επιλέγοντας Ενεργοποίση πύλης πληρωμών. Σε περίπτωση που επιθυμείτε να κάνετε κάποιες δοκιμαστικές αγορές για να ελέγξετε την ορθή λειτουργία του συστήματος σας, μπορείτε να ενεργοποιήσετε το Δοκιμαστικό περιβάλλον.

Έπειτα, θα προσθέσετε τα στοιχεία που θα παρουσιάζονται στον πελάτη σας κατά την ολοκλήρωση της παραγγελίας του, όπως ο τίτλος και η περιγραφή της συγκεκριμένης υπηρεσίας πληρωμής.

Επιπλέον, θα προσθέσετε τα προσωπικά σας στοιχεία, τα οποία θα λάβετε από τη Snappi. Στη συνέχεια θα πρέπει να αποδέχεστε τις πληρωμές μέσα από το developer's portal της Snappi.

== Installation ==

Με την επιτυχή αγορά του πρόσθετου Snappi Pay Later θα δημιουργηθεί ένας λογαριασμός χρήστη, απο όπου μπορείτε να κατεβάσετε το συμπιεσμένο αρχείο σε μορφή zip. Αποθηκεύστε το στον υπολογιστή σας για να ξεκινήσετε στη διαδικασία εγκατάστασης του. Εάν θέλετε μπορούμε να σας το εγκαταστήσουμε και εμείς, για δική σας ευκολία. Τα βήματα εγκατάστασης είναι τα εξής:

1. Από το μενού διαχείρισης επιλέγετε Πρόσθετα, και συγκεκριμένα Νέο Πρόσθετο.
2. Εκεί θα εντοπίσετε πάνω αριστερά το κουμπί Μεταφόρτωση αρχείου. Μόλις το πατήσετε θα εμφανιστεί η φόρμα προσθήκης αρχείου.
3. Επιλέγοντας το κουμπί Επιλογή αρχείου, θα σας εμφανιστεί ένα παράθυρο. Σε αυτό εντοπίστε το zip αρχείο, που αποθηκεύσατε προηγουμένως, και πατήστε το κουμπί Εγκατάσταση τώρα.
4. Μόλις το πρόσθετο ανέβει με επιτυχία στην ιστοσελίδα σας θα σας ζητηθεί να το ενεργοποιήσετε. Σε αυτό το σημείο θα επιλέξετε το κουμπί ενεργοποίησης και το προσθετό σας έχει εγκατασταθεί και ενεργοποιηθεί με επιτυχία.
5. Θα πρέπει να καταχωρήσετε τα εξής στοιχεία στη Snappi

 - Website url: http(s)://www.domain.gr/
 - Referrer page: http(s)://www.domain.gr/checkout/
 - Success Callback/Redirect URL: http(s)://www.domain.gr/wc-api/WC_Snappi_Gateway?action=success&id={orderIdentifier}
 - Fail Callback/Redirect URL: http(s)://www.domain.gr/wc-api/WC_Snappi_Gateway?action=fail&id={orderIdentifier}
